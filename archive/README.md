## Legacy Static Site Archive

This tree preserves the pre-Astro Threatpedia static-site artifacts that used
to live at the repository root.

The live site now builds from `site/` and is hosted at
[threatpedia.wiki](https://threatpedia.wiki).

These archived files are kept only for historical reference and should not be
used as the active source of truth for content, navigation, pipeline state, or
deployment behavior.
